# \# YMVISOR 👁️

# > \*\*Un visor de pantalla completa minimalista y de alto rendimiento para capturadoras de video HDMI\*\*

# 

# \---

# 

# \*\*YMVISOR\*\* es una aplicación ultra-ligera diseñada específicamente para proyectar la señal de tus capturadoras de video HDMI directamente en tu pantalla sin retrasos ni configuraciones complicadas. 

# 

# Si solo quieres ver la señal de un dispositivo externo en tu computadora de forma inmediata y limpia, este programa elimina por completo la necesidad de usar softwares pesados, lentos o llenos de menús innecesarios como OBS Studio.

# 

# \---

# 

# \## ✨ Características Principales

# 

# \* \*\*📺 Pantalla Completa Pura:\*\* Arranca directamente mostrando el video limpio, sin bordes de ventana, barras de tareas ni menús estorbosos. Todo el espacio es para tu video.

# \* \*\*🔊 Audio Integrado Independiente:\*\* Te permite escuchar el sonido que viene del cable HDMI directo en tus bocinas o audífonos en tiempo real, con control total.

# \* \*\*💾 Memoria Automática:\*\* El programa recuerda perfectamente qué capturadora, resolución y fluidez usaste la última vez para que la próxima vez inicies con un solo clic.

# \* \*\*🎨 Interfaz Oscura Confortable:\*\* Un panel de inicio moderno, limpio y oscuro que no cansa la vista cuando juegas o trabajas de noche.

# \* \*\*⌨️ Control Total con Atajos:\*\* Controla todo el programa de manera instantánea usando solo tres teclas (`ESC` para salir, `F` para cambiar el tamaño y `M` para volver al menú).

# 

# \---

# 

# \## 🎮 ¿Para qué se usa? (Casos de uso ideales)

# 

# \* \*\*Jugar Consolas en tu Laptop:\*\* Conecta tu PlayStation, Xbox, Nintendo Switch o consolas retro a tu capturadora y conviértete en una pantalla portátil para jugar en cualquier lugar sin retardo (\*input lag\*).

# \* \*\*Monitoreo de Cámaras Profesionales:\*\* Usa la pantalla de tu computadora como un monitor gigante para ver lo que encuadra tu cámara DSLR, cámara de acción o videocámara en tiempo real.

# \* \*\*Transmitir y Revisar Dispositivos Externos:\*\* Ideal para técnicos, desarrolladores de hardware o creadores que necesitan visualizar menús de otros equipos, decodificadores de TV o computadoras secundarias de manera inmediata.
